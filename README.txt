List of third-party libraries
====================================================
	•	Retrofit  turns your HTTP API into a Java interface.
	•	Picasso allows for hassle-free image loading in your application—often in one line of code!
	•	Dagger2 is a fully static, compile-time dependency injection framework for both Java and Android.
	•	JUnit is a simple framework to write repeatable tests.
=========================================================
What are the reasons to use these libraries?

Retrofit was used to create an interface that allow the TMDb App to communicate with the TMDB Api.
Picasso library easily manages images from the internet. It allows the app to load and transform the movie’s images.
Dagger2 implements an effective way to inject dependencies. It’s possible to create an modularization of App’s MovieAPI.
JUnit creates unit tests that ensure the application’s integrity.
